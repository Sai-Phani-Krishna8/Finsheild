PAN Card Format Guide

AAA T S 9999 Z
Characters 1–3: Series code
Character 4: Taxpayer type (P/C/H/F/A/T/B/L/J/G)
Character 5: First letter of surname
Characters 6–9: Sequential number
Character 10: Check digit
